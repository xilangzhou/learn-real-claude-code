# アーキテクチャ概要

このプロジェクトは、大きな coding agent を一行ずつ真似るものではなく、より小さな Python runtime を教えるためのものです。その機構境界は研究対象の実装に合わせて修正されています。

## このコースが最適化していること

- 表面的な類似より機構の正確さ
- 製品級複雑さより教育上の明快さ
- 一度に一つのランタイム関心事だけを導入する学習順序

## 五つの層

- ツールと実行: モデルが外界に触る方法
- 計画と協調: 作業をどう組み立て委任するか
- 記憶管理: 長いセッションで筋道を保つ方法
- 並行性: 遅い仕事を前景 loop を止めずに進める方法
- 協働: 複数 worker が state、protocol、isolation を共有する方法

## リポジトリの読み方

- `agents/` は章ごとの Python 教材実装
- `docs/` は三言語の本文
- `web/` はカリキュラム、比較、可視化、ソース表示
- `tests/` は小さな回帰安全網
