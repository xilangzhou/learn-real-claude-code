# 架构总览

这个项目现在不再追求逐行模仿一个大型 coding agent，而是教授一个更小的 Python runtime。它的机制边界已经按照我们研究过的源码重新校准过。

## 这套课程真正优化的是什么

- 机制准确性优先于表面相似
- 教学清晰性优先于产品级复杂度
- 学习路径一次只引入一个新的运行时关注点

## 五个层次

- 工具与执行：模型怎样触达外部世界
- 规划与协调：工作如何被组织与委派
- 记忆管理：长会话怎样保持主线
- 并发：慢任务如何不阻塞前台 loop
- 协作：多个 worker 怎样共享状态、协议和隔离通道

## 这个仓库该怎么读

- `agents/` 放章节级 Python 教学实现
- `docs/` 放三种语言的章节正文
- `web/` 负责课程页面、源码视图、对比和可视化
- `tests/` 提供一层小而有用的回归保护
